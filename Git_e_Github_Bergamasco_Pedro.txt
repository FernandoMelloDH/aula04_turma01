Mochila do Viajante

1. Quem inventou o sistema de controle de versão Git e por quê?
O inventor do Git foi Linus Torvalds, que não estava satisfeito com outras soluções de controle de versão de código.

2. Quem atualmente é o dono do Github e por quê?
A empresa proprietária do Github é a Microsoft.

3. Existe outra maneira, além do terminal, de trabalhar com o Github?
Sim, por meio de editores de texto (VS Code, Notepad, etc) e interfaces visuais.
